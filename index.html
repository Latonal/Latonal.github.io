Mes projets

Favs

Objective that I had already set myself when I was younger, a customizable homepage, currently still in development but the first version if already accessible with the most basic features available.

A project that seems simple but a very heavy code aimed at being optimized, both in terms of its length and on the side of your browser. My ambition was to make a light site in order to oppose the sites that were much too heavy even at their generation, which is today a standard that is far too present.

Made with HTML/CSS and vanilla Javascript (without library). For the database, I used IndexedDB which is a database saving the informations in your browser itself, so there is no cost on my side and close to no impact on yours given that the memory space required is negligible.

This first version was quite a big challenge, and much more is to come: page system, addition of customization of icons, facilitation to modify part of the CSS...

  • Create a database architecture (JSON for local use) and manage its use based on user actions
  • Creation of a tool with the aim of making it evolve over time
  • Creation of the front and back architecture and their optimization following the functional development model pattern
